# Morse Code Translator
간단 모스 부호 변환 장치

사용 방법: 
  1. 모스 부호를 텍스트로 변환하고 싶다면, morse_to_text.py를 실행한다. 
  2. 텍스트를 모스 부호로 변환하고 싶다면, text_to_morse.py를 실행한다.
  3. morse_code_core.py는 건드리지 말자. 수정해봤자 도움이 될 것은 없다.
